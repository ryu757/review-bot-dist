# Google レビュー自動返信システム

Google マップのレビューを自動取得し、AI が返信を下書き → スプレッドシートで確認 → 「投稿する」に変えるだけで Google マップに自動投稿されます。

返信は**ホームページの内容を参照して生成**されるため、書かれていない情報を捏造しません。

- 自社ホームページの URL（あれば。AI が参照します）

# 🛠 セットアップの全体像（合計 30〜40 分）

実際の操作順に **4 段階** で進めます：

| 段階 | 内容 | 所要 |
|---|---|---|
| **段階 1** | Google Cloud で OAuth クライアント JSON を作成 | 15〜20 分 |
| **段階 2** | セットアップウィザードを起動 | 5 分 |
| **段階 3** | スプレッドシートに 4 項目を入力（途中で Anthropic API キー取得） | 5〜10 分 |
| **段階 4** | Google ビジネスプロフィールにログイン | 2 分 |

---

## 段階 1：Google Cloud で OAuth クライアントを作成（15〜20 分）

スプレッドシートにアクセスする許可をシステムに与えるための準備です。

### ⚠️ 個人 Google アカウント（@gmail.com）を推奨

会社の Google Workspace アカウントだと **「請求先アカウント必須」「親リソース必須」** といったポリシーで止まることがあります。可能なら個人 Gmail で進めるのがスムーズです。

### 1-1. プロジェクトを作る

1. <https://console.cloud.google.com/projectcreate> を開く
2. **プロジェクト名**: 何でも OK（例: `review-bot`）
3. 「組織」欄が表示されていたら：ドロップダウンを開き **「組織なし」** を選択
   → これで「請求先アカウント」「親リソース」の必須マークが消えます
4. **「作成」** をクリック → 30 秒ほど待つ

> 📷 **[スクリーンショット枠]**

### 1-2. Google Sheets API を有効化

1. 画面上部の **検索バー** に **「Google Sheets API」** と入力
2. 検索結果から **「Google Sheets API」** をクリック
3. 青い **「有効にする」** ボタンをクリック → 数秒待つ

> 📷 **[スクリーンショット枠]**

### 1-3. OAuth 同意画面を構成

> 注：Google Cloud の UI は頻繁に変わります。表示が違っても、似た意味のボタンを探してください。

1. 左メニュー **「APIとサービス」** → **「OAuth 同意画面」**
2. 「Google Auth Platform はまだ構成されていません」が出たら、青い **「開始」** をクリック
3. 順に入力：
   - **アプリ名**: 何でも OK（例: `Review Reply Bot`）
   - **ユーザーサポートメール**: 自分のメールアドレス
   - **オーディエンス（対象）**: **「外部」** を選択
   - **デベロッパーの連絡先**: 自分のメールアドレス
4. **「Google API サービス利用規約に同意する」** にチェック → **「続行」** または **「作成」**

### 1-4. テストユーザーを追加（重要）

設定後、左メニュー **「対象」**（または「テストユーザー」）をクリック

1. **「+ Add users」** または **「+ ユーザーを追加」** をクリック
2. **自分のメールアドレス** を入力 → **「追加」** → **「保存」**

> ⚠️ ここで自分のメールを追加しないと、後で OAuth 認証時に「未承認のアプリ」エラーで進めません。

### 1-5. OAuth クライアント ID を作成

1. 左メニュー **「クライアント」**（または「APIとサービス」→「認証情報」）
2. **「+ クライアントを作成」** または **「+ 認証情報を作成」→「OAuth クライアント ID」**
3. **アプリケーションの種類**: **「デスクトップアプリ」** を選択
4. **名前**: 何でも OK（例: `desktop`）→ **「作成」**

### 1-6. JSON をダウンロード

1. 作成完了後、画面右側または下部の **「JSON をダウンロード」** をクリック
2. ダウンロードフォルダに `client_secret_xxxxx.json` が保存されます
3. **ファイル名や場所は変えなくて OK**（次の段階のウィザードが自動で見つけます）

---

## 段階 2：セットアップウィザードを起動（5 分）

### 2-1. 【Mac の方のみ】初回だけブロック解除コマンドを実行

> ⚠️ Mac の方は **必ず最初にこの作業をしてください**（飛ばすと「開発元が未確認」エラーで進めません）。
> Windows / Linux の方は飛ばして 2-2 へ。

1. キーボード `Command + Space` を押して、「**ターミナル**」と入力 → Enter
2. ターミナルに以下をコピペ（末尾に**半角スペース1つ**を残す）:
   ```
   xattr -dr com.apple.quarantine 
   ```
3. **Finder で解凍したフォルダ**（`Googleマップ自動返信（テスト）` 等）を **ターミナルウィンドウへドラッグ&ドロップ** → パスが自動入力される
4. **Enter** を押す（何も表示されなければ成功）

これで以降は普通にダブルクリックで起動できます。

#### もし上記でうまくいかない場合（代替手順）

`② Mac セットアップ.command` を普通にダブルクリック → 警告 → **アップルメニュー → システム設定 → プライバシーとセキュリティ** → 下にスクロール → **「このまま開く」** をクリック → 認証

### 2-2. セットアップを起動

お使いの OS のファイルを**ダブルクリック**：

- **Mac**: `② Mac セットアップ.command`
- **Windows**: `② Windows セットアップ.bat`
- **Linux**: `② Linux セットアップ.sh`

ターミナルが開き、ウィザードが自動で進みます。Python 環境構築・依存ライブラリインストール・OAuth クライアント JSON 自動検出・Google アカウント認可 まで自動実行。途中で **Y** や **ENTER** で進めてください。

しばらく待つと **ブラウザでスプレッドシートが自動的に開きます** → 段階 3 へ。

---

## 段階 3：スプレッドシートに 4 項目を入力（5〜10 分）

ブラウザが開いた **スプレッドシートの「設定」タブ** に、以下 4 項目を入力します。

| 項目 | 例 |
|---|---|
| 企業名 | 鈴木自動車整備工場 |
| 業界 | 自動車整備 |
| ホームページURL | https://example.com |
| Anthropic APIキー | sk-ant-... |

### 3-1. 企業名 / 業界 / ホームページURL を入力

そのまま入力してください。

### 3-2. Anthropic API キーを取得して入力

API キー欄だけ別途取得が必要です。**別タブで Anthropic を開いて取得 → コピー → 貼り付け** しましょう。

#### 取得手順（5 分）

1. <https://console.anthropic.com> をブラウザで開く
2. **アカウント作成**（既にあればログイン）
3. **クレジットを入金**（必須）：
   - 左メニュー **「Billing」** → **「Add payment method」** でクレジットカード登録
   - **「Add Credits」** で最低 **$5（約 750 円）** 入金
   - 月数十円程度の使用量なので、$5 で何ヶ月も持ちます
4. **API キーを発行**：
   - 左メニュー **「API Keys」** → **「Create Key」**
   - キー名は何でも OK（例: `review-bot`）→ 「Create Key」
   - 表示された **`sk-ant-api03-...` で始まる長い文字列** をコピー
   - ⚠️ **このキーは一度しか表示されません**

#### スプレッドシートに貼り付け

コピーしたキーをスプレッドシートの「Anthropic APIキー」セルに貼り付け。

### 3-3. ターミナルに戻って Enter

入力完了後、ターミナルに戻って **ENTER** を押すと、ウィザードが入力チェックを行います。
未入力やキー形式違反があれば再入力を求められます。

---

## 段階 4：Google ビジネスプロフィールにログイン（2 分）

ウィザードが自動でブラウザを開きます。**返信を投稿する Google アカウント**でログインしてください。

> ⚠️ ここでログインするアカウントは、**Google ビジネスプロフィールの管理者権限**を持つアカウントです。段階 1 の OAuth で使ったアカウントと同じでも別でも構いません（権限さえあれば）。

「未確認のアプリ」と出た場合は **「詳細」→「（アプリ名）に移動」**（テストユーザーに登録済みなら正常）。

ログイン完了後、ブラウザを閉じれば OK。最後に **「今すぐ実行しますか？」→ Y** を押すと初回 sync が走り、スプレッドシートに最初のレビュー下書きが入ります。

「セットアップ完了！」と表示されたら終了です。

---

## 🔄 日常の使い方

セットアップ後は**何もしなくても自動で動きます**。

1. 30 分ごとに新着レビューを自動取得 → スプレッドシートに下書き追加
2. スプレッドシートを確認 → 必要なら下書きを編集
3. 投稿する行の「**ステータス**」列のセルを **クリック → プルダウンから「投稿する」を選択**
4. 10 分以内に自動投稿 → ステータスが「投稿済み」に変わる

### スプレッドシートの 2 つのタブ

- **「レビュー返信」タブ**: 下書きとステータスの確認・編集
- **「設定」タブ**: 企業名・業界・URL・APIキーの変更（次回 sync から反映）

ホームページの内容は **24 時間ごとに自動再取得** されます。

## 🆘 困ったとき

| 状況 | 対処 |
|---|---|
| Cloud Console で「請求先アカウント必須」エラー | 組織ドロップダウンを **「組織なし」** に変更（または個人 Gmail で作り直し） |
| Google ログイン時「未確認のアプリ」 | **「詳細」→「（アプリ名）に移動」** をクリック（テストユーザー登録済みなら正常） |
| Mac で .command が「開けません」 | 段階 2-1 のブロック解除コマンドを実行、または システム設定 → プライバシーとセキュリティ → **「このまま開く」** |
| Anthropic API キーが「Insufficient credits」 | <https://console.anthropic.com/settings/billing> でクレジット入金 |
| 新着が取れない / 投稿失敗 | `Mac 自動実行の停止.command` → `② Mac セットアップ.command` を再実行（ログイン更新） |
| 自動実行を止めたい | `Mac 自動実行の停止.command`（または Windows 版）をダブルクリック |
| 企業情報を変えたい | スプレッドシート「設定」タブを直接編集 → 次回 sync から反映 |
| ホームページ情報をすぐ更新したい | `システム（触らないでください）/credentials/website_cache.json` を削除 |

---

## ⚠️ 注意

- 自動実行は PC が起動している間だけ動きます（スリープ中は止まり、起動時に自動で catch-up）
- Anthropic API は使用量に応じ課金（レビュー1件あたり約 0.5 円）
- Google ログインセッションは数週間〜数ヶ月で切れます（切れたら再セットアップで再ログイン）
